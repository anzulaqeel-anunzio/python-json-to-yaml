# JSON to YAML Converter

A robust command-line tool to convert JSON files into YAML format. Designed to work with standard Python libraries, avoiding heavy dependencies.

## Features

*   **Recursion-Based**: Handles deeply nested objects and arrays.
*   **Safe Formatting**: Automatically quotes strings with special characters.
*   **Simple**: Just does one thing and does it well.

## Usage

```bash
python run_converter.py [file] [options]
```

### Options

*   `--output`, `-o`: Save the result to a YAML file.

### Examples

**1. Convert and Print**
```bash
python run_converter.py config.json
```

**2. Save to File**
```bash
python run_converter.py data.json -o data.yaml
```

## Requirements

*   Python 3.x
